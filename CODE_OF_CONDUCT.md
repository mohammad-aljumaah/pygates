# Code of Conduct

## Purpose

This project is committed to providing a respectful and inclusive environment.

---

## Expected Behavior

* Be respectful and constructive
* Accept feedback gracefully
* Focus on the best outcome for the project

---

## Unacceptable Behavior

* Harassment or discrimination
* Offensive language
* Personal attacks

---

## Enforcement

Project maintainers are responsible for enforcing this code of conduct.

---

## Contact

For concerns, open an issue or contact the maintainer.
